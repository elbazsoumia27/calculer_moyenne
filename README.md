# Calculer Moyenne

A Python utility for calculating average grades and determining admission status.

## Features

- Calculate the average of 3 grades
- Determine admission status based on average (> 10: Admis, ≤ 10: Ajourné)

## Usage

```bash
python Fonction.py
```

## Testing

Run the test suite:

```bash
pytest
```

## Installation

```bash
pip install -e .
```
